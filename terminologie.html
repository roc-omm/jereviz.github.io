Les styles

Titre 1

Pour les titres de chapitres. Raccourci : Ctrl + Alt + 1

Titre 2

Pour les sous-parties. Raccourci : Ctrl + Alt + 2

Titre 3

Pour les sous-sous-parties. Raccourci : Ctrl + Alt + 3

Normal

Pour le corps du texte. Raccourci : Ctrl + Alt + N

Pourquoi utiliser les styles ? Ils permettent de :

  • ✅ Changer la mise en forme en un clic
  • ✅ Générer une table des matières automatique
  • ✅ Naviguer facilement dans le document
  • ✅ Donner un aspect professionnel
